Connects to the [Weaviate Vector Search Engine API](https://weaviate.io/developers/weaviate/api) from Ballerina.

## Package overview
The `ballerinax/weaviate` package is a [Ballerina](https://ballerina.io/) connector for Weaviate Vector Search Engine API. The connector provide access to the manipulations of weaviate schema, objects and search vectors based on various criterias. 

### Compatibility
|                            | Version                       |
|----------------------------|-------------------------------|
| Ballerina Language         | 2201.4.1 (Swan Lake Update 4) |
| Weaviate Vector Search API | 1.18.0                        |
